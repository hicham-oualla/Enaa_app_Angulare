Guide d'utilisation de l'Application Web pour les Apprenants

Bienvenue sur notre plateforme numérique dédiée à la gestion des absences et retards pour les apprenants de l'École Numérique Ahmed Alhansali.

Étape 1 : Contactez l'Administration

Pour obtenir votre code et votre email de connexion, veuillez contacter l'administration de l'école.

email Email : administartion@ecole-numerique.ma

Étape 2 : Connexion

Une fois que vous avez reçu votre code et email, connectez-vous à votre compte :

login Utilisez le formulaire de connexion pour entrer votre code et votre email.

Étape 3 : Consultez vos Absences et Retards

Après vous être connecté avec succès, vous pouvez consulter vos absences et retards :

assignment Consultez les informations sur vos absences et retards directement sur votre tableau de bord.

Étape 4 : Importez vos Justifications

Importez vos justifications en utilisant les codes couleurs suivants :

check_circle Justification acceptée
check_circle Justification en attente de validation
